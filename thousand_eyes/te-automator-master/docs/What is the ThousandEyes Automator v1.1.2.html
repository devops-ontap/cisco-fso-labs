What is the ThousandEyes Automator?

Introduction

To assist Partners in positioning and deploying ThousandEyes for their Customers, the Channel team at ThousandEyes has built out collateral around 5 key use cases. These use cases were chosen based upon the following criteria:

  1. Commonality - address more common services customers are looking to use ThousandEyes for to gain insight into network and application performance visibility,

  2. Well Defined - have defined architectures which require less time and effort when doing discovery with the customer,

  3. Well Documented - have a well defined test regime which allows for a quicker implementation and thus a quicker time to value.

The collateral is keyed around the ability to design and deploy ThousandEyes to gain visibility into application and network performance in a repeatable, comprehensive way to achieve the greatest value for the customer.

This document is designed to provide an overview of one of the tools, the ThousandEyes Automator, which assists in deploying ThousandEyes in support of these use cases.


What is the ThousandEyes Automator?

The ThousandEyes Automator is a chrome extension that can be installed on Windows and Mac devices with Google Chrome installed. The ThousandEyes Automator intended use is for the deployment of a prescribed set of tests, alert rules, labels, dashboards and reports that have been preconfigured for use with ThousandEyes Cloud and Enterprise agents is support of the recommended key use cases.


Installation of the ThousandEyes Automator

The ThousandEyes Automator can be installed on Windows or Mac devices that have Google Chrome installed. The installation instructions can be found here.


ThousandEyes Use Case Deployment using the ThousandEyes Automator

The ThousandEyes Automator is designed to deploy tests, alert rules, labels, credentials, dashboards and reports that support one of the 5 key use cases listed below:

  • Microsoft 365 End User Experience

  • Microsoft Teams End User Experience

  • Cisco Webex End User Experience

  • Cisco SD-WAN Network Performance Visibility

  • Customer Website User Experience


Please follow the instructions found in the ThousandEyes Automator documentation found here to complete the deployment of any of the ThousandEyes 5 key use cases.
