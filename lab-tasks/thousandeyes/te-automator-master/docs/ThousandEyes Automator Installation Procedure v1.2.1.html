The ThousandEyes Automator Installation Procedure


This document describes the procedure to download and install the ThousandEyes Automator. The use of the ThousandEyes Automator is restricted to Cisco employees and Cisco partners. A Cisco CCOID is required to access and use the ThousandEyes Automator. The ThousandEyes Automator's intended use is in support of Cisco customers in a Proof of Value, a professional services engagement or as part of a full managed service. This document is designed to provide detailed instructions on the installation of the ThousandEyes Automator. Any questions in regard to the use, distribution and installation of the ThousandEyes Automator should be addressed to channel@thousandeyes.com.

What is the Automator?

The ThousandEyes Automator is a chrome extension that can be installed on Windows and Mac devices with Google Chrome installed. The ThousandEyes Automator intended use is for deployment of a prescribed set of tests, alert rules, labels, dashboards and reports that have been preconfigured for use with ThousandEyes Cloud and Enterprise agents is support of the recommended key use cases.


Installing the ThousandEyes Automator

The ThousandEyes Automator is a Chrome Browser extension that can be installed on Windows and Mac devices with Google Chrome installed. To install the Automator follow the directions below:


Pre-requisites

1. A Windows or Mac device with Google Chrome installed

2. Access to Cisco SalesConnect to download the ThousandEyes_Automator.zip file


Downloading the ThousandEyes Automator Zip file

1. Use this link to access the ThousandEyes_Automator.zip file

a. You will be required to login using your CCOID

b. Download the zip file to a local directory (this should be a permanent location) then expand the zip file to create the directory.

Installation Instructions

1. Open Chrome and go to Settings by clicking on the 3 dots in the upper right-hand corner of the browser.


2. Click on Extension


3. Click on the Developer Mode in the upper right-hand corner


4. Click on “Load unpacked”


5. Browse to the directory that was created when you unzipped the ThousandEyes_Automator.zip file and Click on Select.



6. The ThousandEyes Automator will appear in the Chrome Extensions page.


7. Close the Extensions and Settings tab and then click on the Extensions icon in the upper right-hand corner of the browser, scroll down to find the ThousandEyes Automator and Click on the pin icon to pin the extension to the toolbar.




8. Installation Complete
