PARTNER RECOMMENDED KEY USE CASES

Use Case #1 - Microsoft 365 End User Experience

This use case is aligned to the use of ThousandEyes to provide application and network visibility for a Microsoft 365 customer environment. Details on the Microsoft 365 architecture and ThousandEyes deployment methodology can be found in the Microsoft 365 Reference Architecture Guide.

The Microsoft 365 End User Experience use case has 2 options: Comprehensive and Lite

Use Case #4 - Cisco SD-WAN Network Performance Visibility

Introduction

This use case is aligned to the use of ThousandEyes to provide application and network visibility for a Cisco SD-WAN customer environment. The use of the ThousandEyes Automator n support of this use case is limited to the deployment of tests designed to provide insight between a single Branch site and a central location, like a data center following a hub and spoke SD-WAN deployment model. Details on the Cisco SD-WAN architecture and ThousandEyes deployment methodology can be found in the Cisco SD-WAN Reference Architecture Guide.

The Cisco SD-WAN Network Performance Visibility use case has 1 option: Cisco SD-WAN


Use Case #5 - Customer Website User Experience

Introduction

This use case is aligned to the use of ThousandEyes to provide application and network visibility for a Customer-facing website. The goal is to gain visibility into location-centric customer experience. The use of the ThousandEyes Automator in support of this use case would be to create PageLoad tests from ThousandEyes Cloud Agents relative to the customer demographic.
